# 个人中心页

**包含模块**：`UserProfile` (用户信息), `UserSpace` (投稿管理)
**应用场景**：点击头像进入的用户主页、用来展示 UP 主详情

## 1. 个人资料展示 (UserSpaceVO)

展示用户的基本信息以及当前的社交关系状态。

| 字段名 (Field)     | 类型 (Type) | 说明 (Description)                   | 来源 (Source)                 |
| :----------------- | :---------- | :----------------------------------- | :---------------------------- |
| **uid**            | Long        | 用户 ID                              | `t_user_info.user_id`         |
| **nickname**       | String      | 昵称                                 | `t_user_info.nickname`        |
| **avatar**         | String      | 头像 URL                             | `t_user_info.avatar_url`      |
| **sign**           | String      | 个性签名                             | `t_user_info.sign`            |
| **followerCount**  | Long        | 粉丝数                               | `t_user_info.follower_count`  |
| **followingCount** | Long        | 关注数                               | `t_user_info.following_count` |
| **isFollowed**     | Boolean     | **[交互] 我是否关注他** (登录态相关) | 查询 `t_following`            |

---
