# 后台管理页

**包含模块**：`UserTable`, `VideoTable`
**应用场景**：管理员进行内容审核与用户管理。

## 1. 用户管理列表 (AdminUserVO)

展示当前系统内所有注册用户的基础状态。

| 字段名 (Field)   | 类型 (Type) | 说明 (Description)    | 来源 (Source)          |
| :--------------- | :---------- | :-------------------- | :--------------------- |
| **id**           | Long        | 用户 ID               | `t_user.id`            |
| **username**     | String      | 登录账号              | `t_user.username`      |
| **nickname**     | String      | 昵称                  | `t_user_info.nickname` |
| **registerTime** | String      | 注册时间              | `t_user.create_time`   |
| **videoCount**   | Integer     | 发布视频数            | `t_video` (统计)       |
| **status**       | Integer     | 状态 (0:正常, 1:注销) | `t_user.status`        |

---

## 2. 视频管理列表 (AdminVideoVO)

展示当前系统内所有上传的视频。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)    | 来源 (Source)          |
| :------------- | :---------- | :-------------------- | :--------------------- |
| **id**         | Long        | 视频 ID               | `t_video.id`           |
| **title**      | String      | 视频标题              | `t_video.title`        |
| **authorName** | String      | UP主昵称              | `t_user_info.nickname` |
| **viewCount**  | Long        | 播放量                | `t_video.view_count`   |
| **createTime** | String      | 发布时间              | `t_video.create_time`  |
| **status**     | Integer     | 状态 (0:已发布, 1:已下架) | `t_video.status`   |
