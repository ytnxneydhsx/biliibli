# 搜索页

**包含模块**：`SearchBar`, `SearchResultList`
**应用场景**：用户主动查找视频或UP主。

## 1. 搜索结果 (复用现有 VO)

搜索结果通常包含**视频**和**用户**两类。

### A. 视频搜索结果
复用 `首页与列表页.md` 中的 **VideoCardVO**。
- **展示逻辑**：可以高亮匹配的标题文字。

### B. 用户搜索结果 (UserSearchVO)
复用 `个人中心页.md` 中的 **UserTinyVO** 或扩展它。

| 字段名 (Field)    | 类型 (Type) | 说明 (Description)        | 来源 (Source)                |
| :---------------- | :---------- | :------------------------ | :--------------------------- |
| **uid**           | Long        | 用户 ID                   | `t_user_info.user_id`        |
| **nickname**      | String      | 昵称 (高亮匹配)           | `t_user_info.nickname`       |
| **avatar**        | String      | 头像                      | `t_user_info.avatar_url`     |
| **followerCount** | String      | 粉丝数 (例如 "200万粉丝") | `t_user_info.follower_count` |
| **videoCount**    | Integer     | 稿件数                    | `统计`                       |
| **isFollowed**    | Boolean     | **[交互] 我是否关注**     | 查询 `t_following`           |
