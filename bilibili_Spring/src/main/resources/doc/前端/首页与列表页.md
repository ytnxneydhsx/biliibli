# 首页与视频列表页

**包含模块**：`VideoCard` (视频卡片流)
**应用场景**：首页推荐、搜索结果、分区列表

## 1. 视频列表卡片 (VideoCardVO)

该卡片轻量且高效，仅展示吸引用户点击的核心信息。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)            | 来源 (Source)          |
| :------------- | :---------- | :---------------------------- | :--------------------- |
| **id**         | Long        | 视频 ID (用于跳转)            | `t_video.id`           |
| **title**      | String      | 视频标题                      | `t_video.title`        |
| **coverUrl**   | String      | 封面图 URL                    | `t_video.cover_url`    |
| **duration**   | String      | 视频时长 (格式化后 "04:20")   | `t_video.duration`     |
| **authorUid**  | Long        | UP主用户ID（用于跳转主页）        | `t_video.user_id`      |
| **authorName** | String      | UP主昵称                      | `t_user_info.nickname` |
| **createTime** | String      | 发布时间 (相对时间 "2小时前") | `t_video.create_time`  |
| **viewCount**  | String      | 播放量 (格式化 "1.2万")       | `t_video.view_count`   |
