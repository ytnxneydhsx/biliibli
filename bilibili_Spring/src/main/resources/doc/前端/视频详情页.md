# 视频详情页

**包含模块**：`VideoPlayer` (播放器), `VideoInfo` (简介栏), `CommentSection` (评论区), `DanmakuLayer` (弹幕)
**应用场景**：核心播放页

## 1. 视频播放与详情 (VideoDetailVO)

包含视频本身的基础信息以及当前用户的交互状态。

| 字段名 (Field) | 类型 (Type)  | 说明 (Description)               | 来源 (Source)            |
| :------------- | :----------- | :------------------------------- | :----------------------- |
| **id**         | Long         | 视频 ID                          | `t_video.id`             |
| **videoUrl**   | String       | **视频源地址** (MP4/M3U8)        | `t_video.video_url`      |
| **title**      | String       | 视频标题                         | `t_video.title`          |
| **desc**       | String       | 视频简介                         | `t_video.description`    |
| **uploadDate** | String       | 发布日期 (2023-10-01 12:00)      | `t_video.create_time`    |
| **tags**       | List<String> | 标签列表                         | `t_video_tag` -> `t_tag` |
| **author**     | UserTinyVO   | 作者的小卡片 (头像/昵称/ID/签名) | `t_user_info`            |
| **viewCount**  | Long         | **[实时]** 播放量                | `t_video.view_count`     |
| **likeCount**  | Long         | **[实时]** 点赞数                | `t_video.like_count`     |
| **isLiked**    | Boolean      | **[交互] 我是否点赞**            | 查询 `t_video_like`      |
| **isFollowed** | Boolean      | **[交互] 我是否关注作者**        | 查询 `t_following`       |

> **UserTinyVO 结构**：
> - `uid`: Long
> - `nickname`: String
> - `avatar`: String

---

## 2. 评论区 (CommentVO)

视频下方的互动区域，支持二级嵌套。

| 字段名 (Field)    | 类型 (Type) | 说明 (Description)     | 来源 (Source)           |
| :---------------- | :---------- | :--------------------- | :---------------------- |
| **id**            | Long        | 评论 ID                | `t_comment.id`          |
| **content**       | String      | 评论内容               | `t_comment.content`     |
| **createTime**    | String      | 发布时间               | `t_comment.create_time` |
| **userInfo**      | UserTinyVO  | 发送者信息 (头像/昵称) | `t_user_info`           |
| **likeCount**     | Long        | **[实时]** 点赞数      | `t_comment.like_count`  |
| **isLiked**       | Boolean     | **[交互] 我是否点赞**  | 查询 `t_comment_like`   |
| **childComments** | List        | 二级子评论 (结构同上)  | 递归/二次查询           |

---

## 3. 弹幕列表 (DanmakuVO)

播放器上的实时评论展示。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)           | 来源 (Source)         |
| :------------- | :---------- | :--------------------------- | :-------------------- |
| **id**         | Long        | 弹幕 ID                      | `t_danmaku.id`        |
| **content**    | String      | 弹幕文本                     | `t_danmaku.content`   |
| **videoTime**  | Long        | 视频内的出现时间 (毫秒)      | `t_danmaku.show_time` |
| **isSelf**     | Boolean     | 是否是我发的 (用于高亮/删除) | 逻辑判断              |
