# 投稿与创作中心页

**包含模块**：`UploadForm` (上传表单), `VideoManager` (稿件管理)
**应用场景**：UP主上传视频、管理已发布的稿件。

## 1. 视频投稿 (VideoUploadVO)

用户上传视频文件及相关元数据。

| 字段名 (Field)  | 类型 (Type)  | 说明 (Description)      | 来源 (Source) |
| :-------------- | :----------- | :---------------------- | :------------ |
| **file**        | File         | 视频文件实体 (分片上传) | 前端上传      |
| **cover**       | File/String  | 封面图片 (文件或URL)    | 前端上传      |
| **title**       | String       | 视频标题                | 前端输入      |
| **description** | String       | 视频简介                | 前端输入      |
| **tags**        | List<String> | 标签 (限制数量/长度)    | 前端输入      |
| **duration**    | Long         | 视频时长(秒)            | 前端/后端解析 |

---

## 2. 创作中心-稿件管理 (VideoManageVO)

UP主查看自己发布的视频状态（已发布、已下架）。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)          | 来源 (Source)         |
| :------------- | :---------- | :-------------------------- | :-------------------- |
| **id**         | Long        | 视频 ID                     | `t_video.id`          |
| **coverUrl**   | String      | 封面图                      | `t_video.cover_url`   |
| **title**      | String      | 标题                        | `t_video.title`       |
| **createTime** | String      | 投稿时间                    | `t_video.create_time` |
| **status**     | String      | 状态 (已发布/已下架)    | `t_video.status` 映射 |
| **stats**      | Object      | 数据概览 (播放/点赞/评论)   | 统计数据              |
