# 排行榜页

**包含模块**：`RankList`, `RankItem`
**应用场景**：展示全站热门视频（V1 基于 MySQL，V2 可升级 Redis）。

## 1. 热门排行榜 (RankVideoVO)

基于播放量排序的视频列表。虽然结构类似 VideoCardVO，但多了排名和趋势信息。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)              | 来源 (Source)          |
| :------------- | :---------- | :------------------------------ | :--------------------- |
| **rank**       | Integer     | **排名** (1, 2, 3...)           | 后端排序后生成         |
| **id**         | Long        | 视频 ID                         | `t_video.id`           |
| **title**      | String      | 视频标题                        | `t_video.title`        |
| **coverUrl**   | String      | 封面图                          | `t_video.cover_url`    |
| **authorName** | String      | UP主昵称                        | `t_user_info.nickname` |
| **viewCount**  | String      | **播放量** (排序依据)           | `t_video.view_count`   |
| **score**      | Double      | 综合得分 (可选: 播放+点赞+硬币) | 算法计算               |

---

## 2. 分区点击量排行 (CategoryRankVO)

针对特定分区 (如动画、游戏) 的点击量排序。  
> 该能力依赖“分区字段/分区关系表”，属于 V2 扩展，V1 可暂不实现。

| 字段名 (Field) | 类型 (Type) | 说明 (Description) | 来源 (Source)   |
| :------------- | :---------- | :----------------- | :-------------- |
| **rank**       | Integer     | 排名               | 后端排序后生成  |
| **id**         | Long        | 视频 ID            | `t_video.id`    |
| **title**      | String      | 视频标题           | `t_video.title` |
| **viewCount**  | Long        | 播放量             | `t_video.view_count` |
