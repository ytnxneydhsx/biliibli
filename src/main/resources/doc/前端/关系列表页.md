# 关系列表页 (Social List)

**包含模块**：`FollowList` (关注列表), `FansList` (粉丝列表)
**应用场景**：个人中心查看谁关注了我，或我关注了谁。

## 1. 关注/粉丝卡片 (UserRelationVO)

一个轻量级的用户卡片，核心是**“当前这一行用户与我的关系”**。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)  | 来源 (Source)            |
| :------------- | :---------- | :------------------ | :----------------------- |
| **uid**        | Long        | 用户 ID             | `t_user_info.user_id`    |
| **nickname**   | String      | 昵称                | `t_user_info.nickname`   |
| **avatar**     | String      | 头像                | `t_user_info.avatar_url` |
| **sign**       | String      | 个性签名            | `t_user_info.sign`       |
| **relation**   | Integer     | **[交互] 关系状态** | 逻辑计算                 |

> **relation 状态码定义**：
> - `0`: 无关系 (Stranger)
> - `1`: 我已关注他 (Followed)
> - `2`: 互相关注 (Friend / Mutual)
