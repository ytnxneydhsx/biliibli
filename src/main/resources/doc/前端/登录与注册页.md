# 登录与注册页

**包含模块**：`LoginForm`, `RegisterForm`
**应用场景**：用户未登录时访问受限资源或主动点击登录按钮。

## 1. 登录 (LoginVO)

用户输入账号和密码进行身份验证。

| 字段名 (Field) | 类型 (Type) | 说明 (Description)          | 来源 (Source) |
| :------------- | :---------- | :-------------------------- | :------------ |
| **username**   | String      | 登录账号                    | 前端输入      |
| **password**   | String      | 密码 (加密传输)             | 前端输入      |
| **captcha**    | String      | 验证码 (可选)               | 前端输入      |
| **token**      | String      | **[响应]** 登录凭证 (V2 可扩展为 JWT)   | 后端生成      |
| **userInfo**   | UserTinyVO  | **[响应]** 登录后的简要信息 | `t_user_info` |

---

## 2. 注册 (RegisterVO)

用户创建新账号。

| 字段名 (Field)      | 类型 (Type) | 说明 (Description) | 来源 (Source) |
| :------------------ | :---------- | :----------------- | :------------ |
| **username**        | String      | 登录账号（唯一）   | 前端输入      |
| **nickname**        | String      | 昵称               | 前端输入      |
| **password**        | String      | 密码               | 前端输入      |
| **confirmPassword** | String      | 确认密码           | 前端输入      |
